watchparty extension

Sync video playback on any streaming site

How to install

1
Download or clone the watchparty project — the extension is in the extension/ folder
2
Open chrome://extensions in Chrome and enable Developer mode (top right)
3
Click Load unpacked and select the extension folder
4
Click the watchparty icon in your toolbar, enter your room code, and click Connect
Extension files: manifest.json · background.js · content.js · popup.html · popup.js
